# 🏆 Smart Banking Transaction Analytics Dashboard (Power BI Project)


An end-to-end data analytics project that explores **banking transaction data across cities, banks, customers, and payment behaviors**, transforming raw financial data into actionable insights.

Built using **Power BI and Excel**, this dashboard uncovers trends in **transactions, balances, payment methods, demographics, and monthly patterns**, with advanced interactivity using **bookmark navigation**.

## 📊 Key Insights

### 🏦 City-wise Performance
- **Mumbai** shows consistently high transaction values and balances  
- **Delhi & Bangalore** show fluctuating transaction patterns  
- Multi-currency handling (INR, USD, EUR, GBP)  

👉 Indicates **regional transaction diversity**

---

### 📈 Monthly Trends
- Peak transactions in:
  - **May (~1.707M)**
  - **October (~1.691M)**  

- Lowest activity:
  - **August (~1.599M)**  

👉 Suggests **seasonality & user behavior patterns**

---

### 💳 Payment & Transaction Behavior
- Filters include:

## 🔍 Workflow

1. **Data Collection**
   - Imported banking transaction dataset

2. **Data Cleaning & Transformation**
   - Standardized columns  
   - Cleaned missing/null values  
   - Converted data types  

3. **Data Modeling**
   - Built relationships across:
     - Transactions  
     - Customers  
     - Banks  

4. **Visualization (Power BI)**
   - KPI Tables

5. **Advanced Features**
   - Bookmark Navigator  
   - Dynamic visual switching  
   - Multi-filter dashboard
